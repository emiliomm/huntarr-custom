Getting Help

  1. Check the FAQ: Most common issues are covered in the FAQ
  2. Check GitHub Issues: Your bug may already be reported
  3. Open a GitHub Issue: For bugs or feature requests

Contributing

Huntarr is open source and welcomes contributions:

  • Bug reports and feature requests via GitHub Issues
  • Code contributions via Pull Requests
  • Documentation improvements

Reporting Bugs

When reporting a bug, include:

  • Huntarr version (shown in the UI footer)
  • Docker or source installation
  • Steps to reproduce the issue
  • Relevant log entries (Settings > Logs, set to DEBUG)
  • Screenshots if applicable
