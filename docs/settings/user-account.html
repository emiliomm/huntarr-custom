User Account

Manage your admin account, security, and authentication settings.

Account Settings

SettingDescription
UsernameYour admin username
Change PasswordUpdate your login password
Recovery KeyView or regenerate your recovery key

Two-Factor Authentication (2FA)

  • Enable 2FA for an extra layer of security
  • Scan the QR code with an authenticator app
  • Enter the verification code to activate
  • You'll need the code for every login

Setting Up 2FA

  1. Go to Settings > User Account
  2. Click "Enable 2FA"
  3. Scan the QR code with your authenticator app
  4. Enter the 6-digit code to verify
  5. Save your backup codes securely
2FA and backup codes

If you lose access to your authenticator app and don't have backup codes, you'll need to use your recovery key to regain access.

Plex SSO

  • Link your Plex account for single sign-on
  • Users can log in with their Plex credentials
  • The admin account remains accessible with username/password

Linking Plex

  1. Click "Link Plex Account" in User Account settings
  2. Log in to Plex when prompted
  3. Authorize Huntarr
  4. Your Plex account is now linked

Recovery Key

  • Generated during initial setup
  • The only way to recover access if you forget your password AND lose 2FA
  • Store it securely (password manager, safe location)
  • Can be viewed in Settings > User Account
Your recovery key is your last resort

If you lose both your password and recovery key, you must delete the database and start fresh.
